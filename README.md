# forgeai-rs

Public, provider-agnostic Rust GenAI SDK workspace.

## Crates

- `forgeai`: high-level SDK entrypoint.
- `forgeai-core`: core types and adapter traits.
- `forgeai-adapter-openai`: OpenAI adapter (starter).
- Additional crates are scaffolded for streaming, tools, schema, routing, observability, replay, and gateway.

## Status

Initial open-source scaffold for `0.1.0`.
